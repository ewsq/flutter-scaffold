## 概述

一个`Flutter`项目脚手架，内含基础模块：

- 底部导航；
- 页面路由；
- 自定义字体图标；
- 网络请求，本地数据库，本地个性化存储等基础封装类；
- UI基础库：`actionsheet`,`alert`,`confirm`等基础交互封装；
- 集成本地总线，极光推送等常用功能；
